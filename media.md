# Медиа-файлы фронтенда

`quests/frontend/src/media/` содержит все статические изображения, которые импортируются компонентами React. Файлы загружаются через обычные `import ... from "../media/..."`, поэтому названия должны совпадать в точности — иначе сборщик не найдёт ресурс.

| Имя файла | Где используется | Что должно находиться в файле |
| --- | --- | --- |
| `background_for_title.gif` | `src/components/title-of-landing.js` | Фоновый GIF для первого экрана лендинга. Анимация должна быть широкоформатной (минимум 1920×1080), без резких цветов; desktop-версия растягивает картинку на всю ширину, а mobile использует её как cover на весь экран. |
| `logo.jpg` | `src/components/header.js` | Квадратный логотип проекта для шапки сайта. Компонент обрезает изображение кругом `h-14 w-14`, поэтому лучше подготовить чистый логотип на однотонном фоне или прозрачный PNG, сохранённый как JPEG. |
| `about_us_1.jpg`, `about_us_2.jpg`, `about_us_3.jpg` | `src/components/about-us.js` | Три карточки блока «О нас». Подготовьте горизонтальные фотографии (минимум 1200 px по длинной стороне), отражающие атмосферу компании/квестов. Нумерация строго соответствует порядку отображения сверху вниз. |
| `service_logo.svg` | `src/components/service-detail.js` | Векторная пиктограмма (12–16 px) рядом с описанием условий участия. Желательно минималистичная иконка, которая хорошо смотрится на тёмном фоне. |
| `telegram.png`, `vk.png`, `whatsapp.png` | `src/components/contacts.js` | Кнопки-ссылки на соцсети и мессенджеры. Графика должна быть квадратной (64–128 px) и с прозрачным фоном, чтобы при наведении масштабирование не искажало лого. |

> Примечание: если добавить новый медиа-файл, его нужно положить в ту же папку и импортировать в соответствующий компонент, иначе bundler не включит его в сборку.
